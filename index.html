        How to use

        1. Base video — the main video that plays after the hook ends. Drop it in the BASE VIDEO box.
        2. Hooks — drop all your hook clips at once. You'll get one output file per hook.
        3. Hit Stitch All and wait. Each job shows its own progress bar.
        4. Download files individually or use Download All once they're done.

        Output files are named basename_hook_1.mp4, _hook_2.mp4, etc. Hook and base video should be the same resolution.
